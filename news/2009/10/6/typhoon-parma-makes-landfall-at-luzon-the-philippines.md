#  Typhoon Parma makes landfall at Luzon, the Philippines. 

MANILA - Weather bureau PAGASA said Tuesday that tropical storm &quot;Pepeng&quot; (international code name Parma) has started to move inland again towards the northwesternmost part of Luzon, as typhoon &quot;Quedan&quot; (international code name Melor) still moves its way towards Okinawa in Japan.
